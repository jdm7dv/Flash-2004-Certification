Making a double-click button
Assigning an action to a button On (Press) or On (Release) is pretty easy, but both of these involve a single-click on the part of the user. For my module, I wanted them to be collapsible when the handle was double-clicked, much like in the MacOS and other similar user interfaces. Creating a double-click button isn't as easy as a single-click button, but it's surprisingly simple.

To accomplish this, you must first embed the handle button in your draggable movie in another movie clip. The hierarchy should look something like this:

    parent movie clip (module)  »  child (new)  »  handle button

This way the handle button will be capable of having multiple states. If you think about how double-click works, it goes something like this:

  • step 1 - press & release (first click)
  • step 2 - system waits for possible second click
  • step 3 - if it times out, nothing happens
    step 3 - if there's a second click (press), then do double-click action

Step 2 above, at the system level, is usually assigned in the system's mouse control settings. Some fast users prefer the double-click time-out to be quick, and some slower users like a little more time to get that second click in there.

In order to recreate this timeout process, the child movie clip that contains the handle button will have timeline which contains all three said steps. Here's what the timeline for mine looks like:


Movie clip timeline for time-out and double-click action

In frame one, we have a Stop action. This is the default rest state of the handle button. In the "button" layer I have the draggable button from the first part of this lesson. Frames 2 through 8 represent my time-out for the double click. My movie's properties are set to output this movie at 12 fps, so 6 frames equals a half-second time-out. During the time-out, the same draggable button from the first frame is used, but has different actions assigned to it. If it is clicked during this time-out segment, the double-click actions are then taken.

To make this time-out work, first modify the action assigned to the button at it's rest state. In the "On (Release)" action, after the "Stop Drag" command, add a "Go to and Play" action, and tell the movie clip to play the second frame.

Next you'll need to set the actions for the button while it is in its time-out segment. This time, you'll only need to assign an action for the "On (Press)" mouse event, as shown below:


On (Press) action for double-click button

In my case, when I double-click on my handle I want the module to go to and play a label called "minimize" in my "/featuremod" movie clip, which will then collapse the module into it's minimized state. In the minimized state, I then have a very similar copy of this movie clip for a handle, where the user can drag around the minimized module, but when double-clicked, it is directed to play a label called "maximize" to restore the module to it's opened state.

We're all done! Next, we'll see what the final product looks like.
