How to go to a frame in another scene from a movie clip or loaded movie

Product: Flash Platform: All Versions: 3 and above ID: 14123

Scene Navigation
Most Flash movies are created using a combination of scenes and movie clips. The actions in Flash are the methods used to navigate or communicate between the separate elements in the complex movie.

When sending actions from one timeline to another - to a particular scene or a frame inside that scene - the Tell Target action needs to be used.

Two steps are necessary when communicating with another timeline:

Use "Tell Target" to navigate from one timeline to another timeline.
Flash movies can contain multiple timelines, often nested within each other. For example, when choosing File > New, the main timeline visible is always _level0. When a movie clip is placed on the main timeline, a second timeline (the movie clip's) is added. The movie clip's timeline plays independently of the main timeline. If you wish one timeline to "talk" to another timeline, it is necessary to use the "Tell Target" action.

Target a frame by label instead of using a number.
"Go To" actions that target a frame by number will not work when targeting a specific scene with Tell Target. Instead, use Go To actions and target frame labels inside the scenes. The frames to be targeted can be given frame labels by choosing Modify > Frame. Referencing the frames by labels rather than by frame number is recommended, because if you add or remove frames from a timeline, the frame numbering may change. Frame labels allow for more flexibility, because they are not dependent on the frame numbering system.

Targeting the Main Timeline:

 

There is another similar method of going to a scene in the main timeline, from a loaded movie:

Use "_level0" as the target. In the example above, a button inside the navMC Movie Clip can tell the main timeline to go to the labeled frame "cat" in scene 2 with the following action: 

On (Press) 
  Begin Tell Target ("_level0/")
    Go to and Stop ("cat")
  End Tell Target
End On

Note: Once you've successfully targeted the other timeline with the Begin Tell Target ("/") line, you can send that timeline any Flash action script commands. Using Go to and Stop is just one example.
