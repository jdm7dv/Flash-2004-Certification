TUTORIAL

Screenshot of the Frames you will create in this tutorial:

  1. The first step is to decide what word you are going to spell.  Pick out your color & font before starting.  After deciding these things create each letter and make it a symbol.  Basically, you should have a symbol created for each letter in your word.

     
  2. After creating the symbols add each letter to it's own layer.  Then align them all so they look like a normal word.
     
  3. At Frame#19 create a keyframe in each layer.  There is a shortcut to do this by highlighting one frame all the way down the timeline and hitting F6.

     
  4. Go back to Frame#1 on the timeline and select all objects in Frame#1. Move all the objects up about ½" to 1".

     
  5. While all the letters are still selected use the Modify > Transform > Flip Horizontal command.  This will flip the word backwards to give it the rotation effect as it comes down later on.

     
  6. Resize all the letters to about 115% and skew them to the left.  To rotate with a percentage go to Transform > Scale & Rotate.  To skew just use the rotate tool and grab the circle that appears at the top of the highlighted text.  A double head left/right arrow should appear.  Pull all the letters to the left.

     
  7. Double-click EACH letter and set the 'Tint' color to the same color as your background.  If it is too difficult to do then you can set the 'Alpha' to zero.  But try to avoid this if possible as it is longer to load.
     
  8. Now set the tweening on each layer.  Double click Frame#1 of every layer and set it to Motion Tween and make sure the Easing is set all the way out (under the tweening tab).   This will give it a more realistic effect.  So basically, when the letters come down they will go faster near the end then at the beggining.
     
  9. Now you need to move the first keyframe of every layer 2 space apart from each other.   You can do this by holding down the Ctrl Key (Command for Mac) and putting your mouse over the frame.  It should appear with side to side arrows.  Just drag it a couple of frame to the right.  So basically in the end result you should have letters that all start about 2 frames as the one before it did.

     
  10. As in Step #9 move all the end keyframes in one notch.   Start with the last letter first and move the  the last frame in by one frame.   Basically all the frames should be indented by 1 or 2 frames from each other.   See the picture below:
