Creating a Mouse Lover

1.Create a new movie (File/New or Ctrl N)

2.Create a graphic, any graphic will work and make it a SYMBOL. Name it Mouse Lover Graphic

3.Create a new movie and name it "Mouse Lover Movie", insert the Mouse Lover Graphic into the movie in the first frame of your timeline. Then double click the frame and give it a STOP instance. Create a Keyframe in frame No 2 and delete the graphic and give this frame a STOP instance aswell.

4.Create a new layer and name it Action Script. Double Click the first frame and select the ACTIONS tab. Now by using the "+" button and using the appropiate actions make your script look like this.

Start Drag (GetProperty (_x,_y), lockcenter)

the above is all one line.

5.Now go back to your main movie and hit Crtl L to birng up yor library box. Drag the "Mouse Lover Movie" into the main movie and double click on the movie and select Definition tab, and in the INSTANCE NAME input "Mouse Lover". That is all for now hit ALT ENTER to see what you have done. = ) . If the "Mouse Lover Movie" follows your mouse when you move it yo got it right, If it doesnt go back and check your script.

Making the ON and OFF buttons

6. Make an OFF and ON button and drag them in to your movie.

7.Double click your ON button and go to the ACTIONS tab, and using the "+" button and apprate actions create the following script.

On (Press)

Begin Tell Target ("/Mouse Lover")

Go to and Stop (1)

End Tell Target

End On

8.Double Click your OFF button and go to the ACTIONS tab, and sing the "+" button and the appreate actions create the following script.

 

On (Press)

Begin Tell Target ("/Mouse Lover")

Go to and Stop (2)

End Tell Target

End On
