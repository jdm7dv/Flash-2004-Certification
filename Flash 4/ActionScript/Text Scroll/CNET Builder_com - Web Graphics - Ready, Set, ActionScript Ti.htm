Have Fun With Text
These two tips help you use text wisely.

Scroll Text Fields the Flash 4.0 Way
Flash 4.0 features text field objects that you can use ActionScript to scroll dynamically. Text fields possess two properties that make scrolling possible: .scroll, which is a property you can retrieve and modify, and which refers to the top line currently displayed in the field. By reading the current .scroll value, you can scroll up or down by rewriting the value as a greater or lesser number. The second property, .maxscroll, is read-only, so it can't be altered. The value of .maxscroll is set by Flash and is equal to the maximum possible value of .scroll--in other words, the line number of the last topmost line displayed when the text reaches its end. You can use the .maxscroll property to calculate the total amount of text displayed in a field, and when to disable any scroll up or scroll down buttons.

In its simplest form, you can scroll text with about three lines of ActionScript. Make a new text field, name it text, and make sure that Multiline, Word Wrap, and Disable Editing are selected in the text field properties. Now use the Set Variable action to set the variable text to a very long string. If it's not long, there's no reason to scroll it! Make two buttons, one for scrolling the text field up and one for scrolling the text field down. In the scrollup button, put the following ActionScript:

On (Release)
Set Variable: "text.scroll" = text.scroll - 1
End On

Put the same ActionScript in the scrolldown button, but set the text.scroll property to text.scroll + 1. You can expand on these ideas with many more features, but this is the foundation for every scrolling text field.

Use Aliased Text
In Flash 3.0, developers couldn't leave text aliased if they wanted to. However, in Flash 4.0, following the steps below, you can give the appearance of aliased text:

    1. Select the Type tool, and make sure the text field modifier is turned on.
    2. Enter your text.
    3. Right-click the text field and select Properties.
    4. Make sure the Disable Editing and Do Not Include Font Outlines boxes are checked.
    5. Uncheck the Draw Border and Background option.
Leaving text aliased is a great option when you are using a small font size or when there is a lot of content on the page; aliased text is much easier to read in these situations. There are a few important things to keep in mind when you are using aliased text, however.
  • When you use editable text, choose a font the user is likely to have installed on his or her system, such as Arial or Verdana. Flash also has three of its own device fonts, _sans, _serif, and _typewriter. When you use these, Flash will select the font on the user's system that most closely resembles the device font you choose. Device fonts are helpful when working with large amounts of aliased text.
  • If you are working with paragraphs, make sure the Multiline and Word Wrap options are selected in the text field properties. The text will display correctly in the authoring environment, but after you publish, the text will all be on one line.
  • If you want viewers to be able to select the text with their cursors (but not edit it), leave the Disable Selection option unchecked in the text field properties. This will add copy/paste functionality to your movies.
