Get Loopy
The Loop action is used to execute a block of ActionScript repeatedly while a condition that is set in the beginning of the loop remains true. An important thing to remember about the Loop action is that the screen is not updated until the loop has finished.

Let's say you want to set up a loop that will move a movie clip to the left by 10 pixels and stop when the clip has moved over by 50 pixels. Well, some ActionScript that looks like this seems as if it should do just that:

Set Variable: "i" = "1"
Loop While (i <= 5)
Set Property ("/clip", X Position) = GetProperty("/clip", _x) - 10
Set Variable: "i" = i + 1
End Loop

Looks like it should work, right? Unfortunately, since Flash doesn't update the screen until the loop completes, this ActionScript would just make the movie clip jump 50 pixels to the left, instead of showing the 10-pixel jumps in between.

To implement this kind of action, you have to make your own loop action using a two-keyframe movie clip and a counter variable. First, set a variable on the main timeline that will act as the counter variable. It will keep track of how many times the looping movie clip has run. Set the variable like this:

Set Variable: "counter" = "1"

Next, create a new movie clip symbol with three blank keyframes. Add a Stop action in the first keyframe, which will prevent the movie clip from running until you tell it to do so. Leave the second frame blank, and double-click the third keyframe to bring up the Frame Properties dialog box, adding the following ActionScript in the Actions tab:

If (/:counter <= 5)
Set Property ("/clip", X Position) = GetProperty("/clip", _x) - 10
Set Variable: "/:counter" = /:counter + 1
Go To and Play (2)
Else
Go To and Stop (1)
End If

Notice how the ActionScript referenced the counter variable; it precedes it with /:. This is because the counter variable is on the main timeline, and you have to give Flash the path to the variable (see "Know Where Your Variables Are" for more info). The code will check to see if the variable counter you set in the main timeline is less than or equal to 5; if it is, move the specified clip over 10 pixels each time the movie clip runs.

Whenever the clip runs, it will move the counter variable up by one. When the counter variable has reached the limit set in the If statement (5 in this case), it will send the clip back to the first frame and will prevent the loop from running. When you're finished, drag an instance of the loop movie clip from the Library onto the stage. Give it an instance name of Looper, say when you want the clip to start its loop, and call it with a Tell Target action:

Begin Tell Target ("/looper")
Go To and Play (2)
End Tell Target"
