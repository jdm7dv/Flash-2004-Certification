Randomize It
You can make your movies more interesting by using the Random() function. This function is also essential for developing games; after all, what fun is a game when everything is always the same?

You can use the Random() function to Tell Target random movie clips, generating random effects and many other possibilities. The Random function in its simplest form would look like this:

Random(5) + 1

That action would generate a random number between 1 and 5. You may be wondering why we added the + 1 to that expression. This is because Random(), by default, starts at 0. If we left that + 1 off, the Random() function would generate a random number between 0 and 4.
