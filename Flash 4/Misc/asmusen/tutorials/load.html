Using "Load Movie"

printable version

Load movie is a great command that allows you to load another Flash movie over the first. This is useful, e.g. if you want to have music on you site - just dump the music in another Flash movie, and load it or unload it through the first movie.

This is how you could create such a music Flash movie.

Step 1. Create a Flash movie, containing only The music, looping.

Step 2. Go to your main Flash movie. Let's say you want the music to start from the beginning. Go to the beginning of your main Flash movie, and to a keyframe, assign a "Load Movie" command.

Step 3. In the "Load Movie" dialog, enter the name of the file you want to load. Probably "music.swf". Below this, there's a dialog for "Level" usually set to "1". This is if you want to load several movies, so it simply marks the stacking order of the movies. We need only to load 1. So let it be "1".

Step 4. Now your music starts when you enter the main Flash movie. If you want it, you can create two buttons, so that when you click on one button, it loads the movie "music.swf" to level 1. And a button that unloads the move at level 1. Now you have your Music ON and OFF buttons!

Voila!
