Macromedia Dreamweaver

Dreamweaver is a WYSIWYG HTML program which means What You See Is What You Get Hyper Text Markup Language program. Whew. It is like working in WordPerfect, only it saves as HTML. It has capability of dynamic HTML, and is also a great tool for people who like to manage their own codes. It can be downloaded in a 30 day trial version at http://www.dreamweaver.com.
