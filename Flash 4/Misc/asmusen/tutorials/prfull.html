Creating Fullscreen movies

printable version

Are you annoyed by those pesky frames? Having problems removing that margin? Well then, this is where you would wan't to look.

Making a page margin of 0:

All browsers:

Make a HTML file containing the following frameset:

<frameset rows="100%,*" border="0" frameborder="0" framespacing ="0">
<frame src="flash.html" marginwidht="0" marginheight="0" noresize scrolling="no">
<frame src="blank.html" noresize scrolling="no">
</frameset>
<noframes></noframes>

This will make a frameset, where the "flash.html" is the movie which you want to show with a margin of 0, and blank.html is an empty HTML file.

4.0 browsers:

Netscape 4 and Explorer 4 has another way of making a margin of 0.

Simply insert this code in the <body> tag of your HTML document:

<body marginwidht="0" marginheight="0" leftmargin="0" topmargin="0" rightmargin="0" bottommargin="0">

Making the Flash movie stretch to fill the browser window: You should definetely use Aftershock to do this. Read my Using Aftershock Utility topic for more info. When in Aftershock, specify the size to be: Width = 100%, Height = 100%.

Flash Movie size:
You also need to set the width and height of flash movies, so here's a list:

640x480 maximized 600x300
800x600 maximized 760x420
1024x768 maximized 955x600

This should do the trick!
