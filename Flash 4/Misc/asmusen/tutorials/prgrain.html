Creating "Filmgrain"

Making Flash movies look like old style movies, has become very popular lately. Actually, i use it on this very page... Click the tiny "m" in a circle on my main movie to toggle...
Actually, it's pretty easy to make, it only takes time.

Step 1. Press F8 to make a movie clip.

Step 2. Select the Pencil, set to Ink, and choose a the color you wish to give the grain (i use gray).

Step 3. Draw a lot of small short lines, which doesn't have any system or symmetry. Not too many, and not too big.

Step 4. Create a new keyframe, by selecting the next frame in the timeline, and pressing F6. Repeat step 3.

Step 5. Repeat steps 3 through 4, untill you feel you have enough to make it look as if the grain doesn't loop.

That's how you create Filmgrain! Easy, but timeconsuming.

Tip: You can also create lines, or small circles, to emphasize that it is an old movie.
