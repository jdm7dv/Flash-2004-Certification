Opening a new browser window

Opening a new browser window, can greatly improve user-friendlyness of your site. Read here, if you want to know how it's done.

Step 1. Insert this JavaScript in the head of your main page:

<script language="JavaScript">
function MM_openBrWindow(theURL,winName,features) { window.open(theURL,winName,features);
}
</script>

Step 2. Now, go to the button, in your Flash movie, which you wan't to give the action to open a browser window. Click "Properties" for it.

Step 3. Assign a "Get URL" command for this button, containging the following:

javascript:MM_openBrWindow('file.html','Windownameinoneord','toolbar=yes,location=yes,status=yes,menubar=yes,scrollbars=yes,resizable=yes,width=300,height=200')

Step 4. Make sure, that when you Aftershock your Flash movie, check the box called "Enable browser scripting and FS Command".

Voila!

NOTE: Use this feature with care, people tend to get annoyed from popup windows. Limit the use...
