Creating 3D

3D in Flash is the latest trend. Flash was never designed for making 3D, but using some techniques it is possible to create scalable 3D.

To create such 3D though, you need an other program than Flash, a 3D program. I would recommend Adobe Dimensions, which can be downloaded in a 30 day free trial version at http://www.adobe.com/.

Adobe Dimensions can export your 3D artwork into the Adobe Illustrator format (.ai) which is a vector file. This is useful, because then you won't have to trace your artwork in Flash, which can be quite time consuming.

Download a sample .fla file.

Step 1. Create your 3D animation in Dimensions. Render to PostScript.

Step 2. Export this artwork as a sequence of .ai files.

Step 3. Dimensions is known to create a lot of useless symbols, when imported into Flash. If you wish to remove these without much trouble, you might want to use Adobe Illustrator 7 > to import the files, and chose "Trim". This will remove the symbols.

Step 4. In Flash, import the first of these files, and Flash will ask you to import the entire sequence. Apply to this.

Step 5. Actually, you now have your 3D animation. Now comes the optimization process. Go over every frame, and break it up as much as possible (CTRL + B). Dimensions tends to create a lot of symbols when exporting .ai files. If you break up every frame, these symbols will become unnecessary and you can delete them. If you used the Illustrator tip in step 3, you should be able to freely break up the symbols.

Step 6. If you used gradients in Dimensions, then every time you break up the artwork in Flash, a gray wireframe will appear. Delete this wireframe. Now, select a gradient that Dimensions made, and replace it with a matching Flash gradient.

There you have it. 3D in Flash.
