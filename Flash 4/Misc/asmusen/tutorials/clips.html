Using Movie Clips

printable version

Movie clips are great if you want to make advanced animations, or create animated buttons. It is easy to create them:

Step 1. Press F8 and choose movie clip.

Step 2. Make an animation, either shape or motion tweening.

Voila!

The problem though, is using them. Movie clips should only be used in the following cases:

  • If you want to animate an already animated symbol.
  • If you need to use Tell Target
  • If you need to create a looping symbol or effect, that is independend of other animation e.g. my snowflakes.

If you need to know more about movie clips, please examine the Tell Target tutorial, which also makes use of Movie Clips.
