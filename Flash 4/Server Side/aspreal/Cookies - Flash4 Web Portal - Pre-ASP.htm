~flashwave: Pre-Tested ASP - PHP3 Chat App


Quick Notes: A very simple little script which *should* help you write your own cookies. Now, all you have to do is send some variables to it. Lets say for instance I want to record my name and my url, I would send the following in flash.

cookies.asp?action=bakemycookie
This says to the script that we wish to create out cookies.

&cookiename=info
This tells the script what we want the overall cookie to be called.

&numcookies=2
This defines how many bits of data we wish to send to our coookie.

&name1=UserName
This defines the name of our first bit of data.

&name2=URL
This defines the second part of our data.

&data1=Tom
This is our data for the first heading - ie - "UserName".

&data2=flashwave.co.uk
This is the data for the second heading - ie - "URL".

Now isn't that easy!? All you havr to do, if you wish to add more cookies, is increase the "numcookies", "name" and "data" variables.
You will need the required files to execute this app, and they can be found here

<%response.buffer=true%>

<%

'Get the populating variables from the HTTP Header and the Server
action = Request("action")

'check status then do what it says
if action = "fetchmycookie" Then
	Fetch_My_Cookie()
elseif action = "bakemycookie" Then
	Bake_My_Cookie()
else
	message = "No functions sent to the script"
	response.write("&message="+URLEncode(message))
	Response.End
end if

function Fetch_My_Cookie ()
cookiename = Request("cookiename")
numcookies = Request("numcookies")


for i = 1 to numcookies
	mycookies = "mycookie"&(i)
	mycookie = GetCookie(cookiename,request("name"&(i)))
        response.write mycookie
next

End Function

function Bake_My_Cookie()
numcookies = request("numcookies")
cookiename = request("cookiename")


for i = 1 to numcookies
Call AddCookie(cookiename, "&", "", 365)
Call AddCookie(cookiename, request("name"&(i)), request("data"&(i)), 365)
Call AddCookie(cookiename, "&", "", 365)
response.write request("name"&(i))
response.write request("data"&(i)) & VbCrlf
next

End Function

%>
