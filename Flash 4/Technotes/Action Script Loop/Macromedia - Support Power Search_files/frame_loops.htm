How can I loop over some frames a given number of times?

Product: Flash Platform: All Versions: 4.0 and above ID: 14023

The ActionScript in Flash 4 makes it easy to count things. You can use a variable to count the number of times you pass through a frame. For instance, if you'd like to repeat frames 10 through 20 five times, then an action like the following on frame 20 can do the trick:

    Set Variable: "counter" = counter + 1
    If (counter < 5)
      Go To and Play (10)
    End If
