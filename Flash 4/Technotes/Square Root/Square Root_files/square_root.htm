Finding square roots in Flash

Product: Flash Platform: All Versions: 4 and above ID: 14250

Finding square roots
This TechNote discusses how to create an ActionScript that will find the square root of a number.

In the Flash movie below, the original value is entered into an editable text field by the user. Pressing the button will yield the square root of the entered number.

To build the square root calculator:
1 Create an editable text field. Select the text field's properties and give it the variable name "value". This will be the entry field for the original value.
2 Create an editable text field to hold the result. Select the text field's properties and give it the variable name "sqrt". This will be the display field for the square root result.
3

Double click the button instance to access the Instance Properties dialog.

Select the Actions tab. Enter the following ActionScript code: 

   On (Release)
          Set Variable: "i" = "0" 
          Set Variable: "x" = "1" 
          Loop While (i < 20) 
            Set Variable: "x" = x-((x*x-value)/(2*x)) 
            Set Variable: "i" = i+1 
            End Loop 
          Set Variable: "sqrt" = x 
  End On

Note: This script uses the variables 'value' and 'sqrt' from steps 1 and 2, which have been colorized for clarity. Be certain all of the "Value" fields in Set Variable actions are set to "Expression".

Additional information
For more detail on working with editable text fields, please refer to Using type blocks and text fields in Flash (TechNote 14154). For other math functions in Flash try Branden's Trig Library from Flashlite.net.
