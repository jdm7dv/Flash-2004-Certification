The FSCommand in Shockwave Flash 3.0

This demonstration cannot be run within a browser. You can only see these specific FSCommands within a standalone projector or the Player.

FSCommand Arguments What It Does
AllowScale True Displays the SWF scaled (not always 100%)
  False SWF is always 100%
ShowMenu True Displays the full Flash menu on right click (Windows) or control click (Macintosh)
  False Displays "About Shockwave Flash 3.0" only (Windows) or grays out the menu (Macintosh)
FullScreen True SWF is displayed full screen
  False SWF is displayed at original size
Exec program name

Executes program name.
This command is valid for standalone projectors only.

Quit   Quits the SWF

To see these FSCommands in action, run the application FSCOMMAND.EXE.
