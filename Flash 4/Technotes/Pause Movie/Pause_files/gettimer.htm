How to use GetTimer to pause a movie

Product: Flash Platform: All Versions: 4 and above ID: 14207

Use GetTimer to measure time
GetTimer is a function in Flash that can be used to check how long the Flash movie has been playing, in milliseconds. Flash's internal time begins counting when the movie begins playing, and increases as long as the movie is playing. The timer cannot be reset while the movie is running. However, using variables and scripting, the GetTimer function makes it possible to synch actions to a specific measure of time.

The example below provides steps that will create a new document with a movie clip that serves as a "timer". The movie clip controls the main timeline, when the specified time has elapsed. To use this technique in an existing movie, apply the logic from this example, integrating the components into the existing project.

Scripting can be confusing for beginners, so it is recommended that users study the available documentation on Flash actions and expressions. Detailed information can be found in the Flash manual, as well as online at the Flash support center. Information on using GetTimer is also available on page 199 of the Flash 4 manual.

This tutorial will construct a pause mechanism using:
Frame actions using Set Variable and the If statement.

A movie clip that acts as a timer.

Prepare the main movie:

Note: This is the beginning of a new movie. It would be helpful to put something on the stage for illustrative purposes, to show that the movie is playing. Adding text to the stage that says "this is our movie" is sufficient.

1

In a new document, add four blank frames to the timeline so that it has a grand total of five frames.

2

In the fifth frame, add a keyframe and assign it the action Stop.

3 After this action has been assigned, assign the action Set Variable. In the Variable field, type stoppedtime. In the Value field, type GetTimer and set this field to "Expression".
Note: All variables in this TechNote (except for GetTimer) can be renamed at the user's discretion. For the purposes of this tutorial, they have been given names that are intuitive and relate to their function in this example.

When asked to type in the name of a variable, do not type in the periods or punctuation used to end sentences in this TechNote.

4

Select the fifth frame of the movie. Choose the text tool from the toolbar and create some text on the stage to alert the user when the movie is stopped.

Make a movie clip that counts time:

5 From the menu, choose Insert > New Symbol and create a movie clip.
Name it "timer clip".
6 In the first frame of the movie clip ("timer clip"), add the action Set Variable. The Variable name should be stoppedtime and the Value should be ..:stoppedtime. Set the Value field to "Expression".
This allows us to check the value of the this variable from the main timeline. Since variables in Flash only exist in their own timeline, we're creating another variable with the same name that will exist inside this movie clip only.

While the name "stoppedtime" is the same, this is referring to two different variables - one in the main timeline and one in the current timeline, which is a movie clip. The additional punctuation "..:" is a way to refer to the variable in the main timeline.
7 In this same menu, add another Set Variable action. The Variable name should be pausetime, and its Value should be the amount of time, in milliseconds, that the movie should pause for. For this example, we've chosen 10000 milliseconds, or ten seconds.
8 In this same menu, add the If action. In the Condition field, write the following:

GetTimer>=pausetime+stoppedtime

This is comparing GetTimer (the current amount of time the movie has been running) with a figure that represents the desired time to elapse before the movie is to play again.
9 Additionally, in this menu, add the action Tell Target. For the Target, enter a forward slash ("/") to indicate the main timeline. Add another action to be inserted between the Tell Target statements - Play.
10 The final action to add to this frame is another Set Variable action. The name of the variable should be counter. In the Value field, type this information exactly:

Int(((pausetime+stoppedtime)-GetTimer)/1000)

Set this field to "Expression". This creates a number that will be equal to the paused time remaining. It will act as a "countdown". The figure is being divided by 1000 to get a number in seconds, and the Int function is being used to round this figure to the nearest integer.

Your actions should look like this:

11 Click OK to close the Frame Properties dialog box.
12

Add one blank frame (not a keyframe) to the timeline, to give this movie clip a total of two frames.

Note: This movie clip must have at least two frames to allow it to loop and repeat the actions in the first frame. It won't work without this step.

13

Choose the text tool on the tool bar, and select the text field modifier button on the bottom of the tool bar. On the stage inside this symbol, add a text field.

14 Once the text field has been created, choose Modify > Text Field from the menu. Assign this text field the variable name counter. Click OK when this is done. This will allow the variable counter to appear in this text field as the movie clip plays.

Return to the main timeline:

15

Exit this movie clip and return to the main timeline. In frame 5 of the main timeline, drag a copy of the timer clip symbol from the library.

16

Add a new keyframe to the timeline in the sixth frame. Delete the timer clip from the stage. Change the text from the previous frame to indicate the movie is now playing.

17 Add a Stop action to the sixth frame to conclude this example.
