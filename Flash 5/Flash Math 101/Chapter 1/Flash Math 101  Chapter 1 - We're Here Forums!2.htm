Chapter 1 - Distance

Our initial exploration begins with the basic tools which the developer requires to map objects on the graphic stage with interactive scripting. There are several math functions utilized when mapping objects, so please explore each link below, and we will combine the functions later into a scripting application.

Syntax:  ActionScript
        Labels and Instances
        Comments: will be used by the script author(s)
        //will be for my comments-->

NEWTON'S APPROXIMATION FOR SQUAREROOT
n = n - ((n^2 - input) / (2 * n))

PYTHAGOREAN THEOREM
a^2 + b^2 = c^2

DISTANCE
hypotenuse = sqrt((x1 - x0)^2 + (y1 - y0)^2)

For interactivity to occur between objects, we must first set the position for each object relative to the other, thus calling distance between objects as the initial function. We utilize the Pythagorean Theorem and Newton's approximation for squareroot in the equation.

This squareroot approximation relies on many iterations for accuracy, which can become somewhat cpu intensive. There are occasions when a comparison between the square of the hypotenuse of two or more sets of objects will suffice, avoiding the costly step of solving for squareroot. I've engineered this efficiency into the evaluating shortest distance script from The Spider vs. The Fly game, which we shall discuss later in this chapter.

Our resident Flashjunkie, Steve Cubbon, who has flooded the tutorial section with a plethora of new source files, has included a plug-n-play squareroot calculator to include in your Flash tool library. This squareroot script is utilized often in more complex programming. Object programming's greatest advantage is the plug-n-play aspect of a block of code, where you set the code once on the actions layer, label it and call it from various frames, layers, MCs or levels to interact with objects as required.
You can download the source .fla here.

Here's the code:

Squareroot Calculator
By: Steve Cubbon

Variables - iterations, n, counter
Textfields - input, output
Buttons - Button
Layers - Squareroot function

Frame 1

Set Variable: "iterations" = 15 //sets loop cycles-->
Stop

Frame 2: label - sqrt

Comment: Newton's Method for Finding Squareroot
Set Variable: "n" = 1 //nominal starting point-->
Set Variable: "counter" = 0 //initializes loop counter-->
Loop While (counter < iterations) //compares present count with maximum-->
      Set Variable: "n" = n - ((n * n - input) / (2 * n)) //Newton's squareroot-->
      Set Variable: "counter" = counter + 1 //adds 1 to loop count-->
End Loop
Set Variable: "/:output" = n //displays final iteration as main timeline output-->

Button

On (Release) //mouse click and release-->
Call ("sqrt") //calls the labelled "sqrt" script-->
End On

Steve has used '15' as the number of iterations to cycle for accuracy. Reviewing the various examples of the squareroot function available finds that this number of cycles is an average value, which should be accurate enough for most of our purposes. As seen in the squareroot expansion notebook, it doesn't take many iterations for a smaller number to be rendered quite accurately, so don't overscript the cycles needlessly.
