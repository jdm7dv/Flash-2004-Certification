Loading an XML file is asynchronous and takes time - certainly longer than the time from one action script command to another.

The correct way to detect when the xml is loaded is to create your own "onLoad" callback function and register it with the XML object. Then when the XML file has been loaded and parsed your function will get called. Depending on where the XML file is coming from that could be anywhere from a few millisecs to a minute or two.

If you must test the "loaded" property then do it in a loop (with a timeout count) to allow time for the XML to be loaded.

I got it to wrok

Thanks for all your replies. Here is what i used to get it to work. The key is the xml.onLoad event. That gets fired after the xml doc is loaded. You overload the onload event with your own. I have this in the first frame of my movie.

doc = new XML();
doc.onLoad = myOnLoad;
doc.load("text.xml");
function myOnLoad(success) {
if (success) {
_root.gotoAndStop("loaded");
} else {
_root.gotoAndStop("notloaded");
}
}
stop();
