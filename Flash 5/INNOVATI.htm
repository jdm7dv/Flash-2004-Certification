Innovative Windows Program Launcher (Flash 5)
    By: Mattam [ homepage ]
  Download
  
For Flash4 & Flash5

- This programme was made since I worked on MAC machine for a past few years and when I saw Neo Planet web browser that was done in FLASH. So I try to create it in FLASH4 by using FS Command to called the .exe file from the hard drive. And at the last step I created a shortcut icon then paste it in the Startup Item folders in the Windows.

- Ok, lets explain it details :

1. First, you must have or make an interface for use on your desktop programme launcher. I use a montage  pictures that was done in Photoshop then import it into FLASH.



2. Now, in FLASH create your own desktop navigation by using buttons, animations, tweening, actionscripts or what ever ...



3. In Flash, double clicks the button (Flash4) or click the button and open the object action control panel in Flash5.

4. Then add an action to the buttons. In Flash4 click the sign and choose "FS Command" and in Flash5 in object click the Basic Action icon and select "FS Command" icon.
Flash4
Flash5

5. Now, open your MS-DOS Prompt program and find which programs would be launch when the buttons is      clicked. For example, lets take a Photoshop application.

6. When you open the MS-DOS Prompt program, type this command (in green text) to change the directory to Program Files directory :

     C:\WINDOWS>
     C:\WINDOWS>cd\
... ENTER
     C:\>
     C:\>cd\progra~1
... ENTER
     C:\Program Files>


7. Open the Program Files directory by type in the command below to search for Photoshop application :

     C:\Program Files>dir/w
... ENTER

     Now find the [ADOBE] directory and enter this command :
     C:\Program Files>cd\
... ENTER
     C:\>               
     C:\>cd\progra~1\adobe
... ENTER
     C:\Program Files>Adobe>            
      
     Then type in this command :
     C:\Program Files>Adobe>dir/w
... ENTER

     Again search for [PHOTOS~1.5] ... since I used Photoshop 5.5, the command will be like so :

     C:\Program Files>Adobe>cd\
... ENTER
     C:\>               
     C:\>cd\progra~1\adobe\photos~1.5
... ENTER
     C:\Program Files>Adobe>Photoshop 5.5>dir/w
... ENTER 


     And the final steps, find your .EXE file extensions :
     C:\Program Files>Adobe>Photoshop 5.5>cd\
... ENTER 
     C:\>
     C:\>cd\progra~1\adobe\photos~1.5
... ENTER
     C:\Program Files>Adobe>Photoshop 5.5>photoshp.exe
... ENTER 

8. When your MS-DOS program start to loading the Photoshop application, your mission is complete and your Photoshop application directory would be :
     c:\progra~1\adobe\photos~1.5\photoshp.exe
     * notice that all the command should be in the lowercase.

9. Now get the commands and put it into the Flash buttons by type into your button's action.

    
     Flash4 FS Command

     
     Flash5 FS Command

10. DONE. Use your own creativity to create movie clips, animations or what ever for your own Desktop Windows Programme Launcher.


ADDITIONAL TIPS
      Add a final touch to your program by KILL THE RIGHT CLICK. To do this, I used eXeScope and edit all the codes to remove all the right functions.
      

      Last but not least, make a shortcut icon of .exe projector and paste it into the Startup Item folders in the Windows\Start Menu\Programs\StartUp directory.

                   


      And the final Windows Programme Launcher ...
